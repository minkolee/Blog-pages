Java Reinforcement 14 CompletableFuture

在Thinking in Java 中, 知道了ExecutorService 的 .submit()方法会接受一个Callable ,返回一个Future对象, 其中包裹着返回值.

而且使用Future对象不会阻塞, 之后可以使用Future.get()来获取其中的值, 这是简单的异步编程. 如果仅仅单独使用Future接口, 只能在一个线程中等待另一个, 无法完成复杂的带有条件的操作.

Java 8 对于Future接口新实现了一个CompletableFuture类, 并且使用了lambda表达式和流的思想
