判断是不是完全平方数

这个也有取巧的方法, int 范围的最大值也就是从0~46340. 开始暴力计算即可.

class Solution {

    public boolean isPerfectSquare(int num) {
        boolean found = false;

        for (int i = 0; i <= 46340; i++) {
            if (num == i * i) {
                found = true;
                break;
            }
            if (i * i > num) {
                found = false;
                break;
            }
        }

        return found;
    }
}
