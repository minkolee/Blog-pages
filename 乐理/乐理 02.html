五度循环圈

五度循环圈是一个非常重要的概念, 对于后边的很多东西都很重要.

所谓五度循环圈, 从C大调开始, C G D A E B=Cb都是相差纯五度, 然后和B相差纯五度的是Gb=F#, 然后是Db=C#, 之后是 Ab , Eb, Bb, F, C 然后又回到C.

这其实也很容易理解, 相差7个半音, 就是偏移7个键, 无论黑白, 一圈12个, 就是84个键, 基本涵盖了一圈.

相距纯五度的音一起响, 仅次于纯一度和纯八度的和谐程度. 所谓和谐程度, 主要是两个方面, 频率比和泛音列的问题, 以后再看.

很多音乐软件都是使用五度圈来制作和弦或者转调. 好吧, 概念先学着, 现在还没有一点概念, 等后边学了FL studio 添加键盘的时候再看.

调式中各音级的名称

以CDEFGAB为例, 可以用罗马数字I到VII来代表各个音级, 除了音级之外, 还有各个独特的名称:

  1. C - 主音
  2. II级音, 即D - 上主音
  3. C下方小二度也就是VII级音即B - 导音, 有很强的达到主音的倾向
  4. C下方小二度 - 导音, 有很强的达到主音的倾向
  5. C上方纯五度, 即G - 属音
  6. G下方的F,是IV级 - 下属音
  7. III - 中音
  8. VI - 下中音

这个音级的名称主要是为了用来学习小调.

自然小调

与大调相同, 小调也是一种特殊的模式.

大调的3, 6 ,7级音降低半音, 就可以得到同大调对应的小调, 比如:

C 大调  C  D  E  F  G  A  B  C
c 小调  C  D  Eb F  G  Ab Bb C

所以大调的模式是 全全半全全全半.

小调的模式是则是 全半全全半全全

小调降的三个音中, 最重要的是E也就是第三个音, 大调中C与E是大三度, 而小调中C与E是小三度. E有个特殊的名称叫做调式特性音级. 即降不降E决定了调式的特性, 只要降了E, 调式就总感觉不如大调明朗与欢快.

关系大小调

自然小调有一个特殊的小调, 就是a小调, 所有的构成键全部是白键, 所以就演奏来说, 演奏a小调与演奏C大调没有任何区别, 没有任何升降音, 全部演奏白键, 只是主音不同.

只有主音不同, 构成音全部相同的大小调, 叫做关系大小调. 称a小调为C大调的关系小调, C大调为a小调的关系大调.

从a小调与C大调的关系可以知道, 关系大小调有如下关系:

  1. 大调下方小三度的音就是关系小调的主音

所以在五线谱中, 小调的调号并没有单独的记法, 而是使用对应关系大调的记号, 因为使用的琴键都是一样的, 所以弹就是了.

详细的大小调对应关系, 可以在wiki的任意一个调的页面内找到, 比如D大调.

自然音程与变化音程

自然音程里边的音程, 是先找主要音, 看是几度, 再看半音数目.

自然音程指的是: 大调或者自然小调中, 任何两个音构成的都是自然音程, 自然音程分四大类:

  1. 纯音程: 纯一度, 纯四度, 纯五度, 纯八度
  2. 大音程: 大二度, 大三度, 大六度, 大七度
  3. 小音程: 小二度, 小三度, 小六度, 小七度
  4. 三全音: 增四度, 减五度

这个实际上是之前的音程内容, 当时已经总结过了.

自然音程之外的, 都是变化音程, 也分为四类:

  1. 增音程: (不含增四度). 纯音程和大音程中, 度数不变, 增加半音, 得到增音程(增音程在之前的自然音程表上找不到, 因为基础音和半音数量的组合不符合自然音程的要求). 例如 F->G# 就是增二度,因为F-G是两度, 但是相差三个半音, 在自然音程表上找不到, 所以不是自然音程. 然而将G#实际上也是Ab, F->Ab 则是自然音程的小三度, 虽然相差依旧是三个半音.
  2. 减音程: (不含减五度). 纯音程和小音程中, 度数不变, 减少一个半音, 就得到了减音程, 同上边一样, 也是在自然音程表上找不到.
  3. 倍增音程: 增音程再增加一个半音, 就得到倍增音程. 先要有增音程的组合, 才谈得上倍增音程. 这里注意有一个重升记号类似一个乘号, Gx 就是将G提高两个半音
  4. 倍减音程: 减音程再减少一个半音, 得到倍减音程. 也是必须先要有一个减音程的组合, 才能有倍减音程, 重减记号是连续的两个b, 比如 Abb 表示将A降低两个半音

这里唯一特殊的就是, 没有减一度. 不能从纯一度得到减一度.

妈呀, 其实就是同一个键的写法不同, 导致出来四种变化音程.

小调的变体

这里的小调指的是自然小调, 每一个自然小调, 都可以变化为两种, 一个是和声小调, 一个是旋律小调.

变化的逻辑如下:

  1. 将小调的VII音升高半音, 就得到和声小调: 
            a 自然小调 A  B  C  D  E  F  G  A
        a 和声小调 A  B  C  D  E  F  G# A
    G并不是A的导音, 因为差2个半音, 升高之后, 就变成了A的导音. 此外F->G#是一个增二度, 这是和声小调的特点.
  2. 将小调的VI和VII音都升高半音, 就得到旋律小调: 
            a 自然小调 A  B  C  D  E  F  G  A
        a 旋律小调 A  B  C  D  E  F#  G# A
    升高之后, F#和G#都是大二度, 增强了最后三个音的和谐性. 此外旋律小调的一大特点是上行和下行不同, 上行的时候是按照标准演奏F#和G#, 但是下行的时候, 依然还是演奏没有升高的F和G

三种小调与大调的关系

这里列出来一个大调与对应小调的关系:

C 标准大调   C  D  E  F  G  A  B  C
c 自然小调   C  D  Eb F  G  Ab Bb C
c 和声小调   C  D  Eb F  G  Ab B  C
c 旋律小调   C  D  Eb F  G  A  B  C

可见如下规律:

  1. 大调只降低第三音, 就得到对应旋律小调
  2. 大调降低三六音, 变成同主音和声小调

旋律小调和和声小调在五线谱里也没有自己的调号. 用的是自然小调的调号, 但自然小调也没有调号, 用的是关系大调的调号. 所以其实还是用的关系大调的调号.

所以很搞笑的就是和声小调和旋律小调的升高需要用临时调号写出来, 不能写在谱号后边. 谱号依然用对应的关系大调.

举个例子: a小调的关系大调是C大调, 现在想写a旋律小调, 则调号应该是a自然小调, a自然小调的调号是C大调, 所以谱号处没有任何升降. 但是按照要求需要升高F和G, 因此需要在F和G处使用临时升号. 但是在下行的时候, 还需要使用还原记号来还原才行.
