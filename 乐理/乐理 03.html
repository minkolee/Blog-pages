音符 Note

五线谱最基本的就是音符记号. 一个空心的叫做全音符, 一个空心带符干的叫做二分音符, 一个实心带符干的叫做四分音符, 实心+符干+一个尾巴的是八分音符, 实心+符干+两个尾巴的是十六分音符.

一个音符圆圆的地方叫做符头=note head, 中间的部分叫做符干=note stem, 弯曲的部分叫做符尾=note flag.

如果有连续相同的音符连在一起, 可以把符尾连接起来变成符杠=note beam. 如果音符在第三线往上, 符干必须连在音符左下而且朝下.否则就要连在音符右侧然后朝上.而符尾不管上下, 一律在符干右侧.

如果同时演奏的音的长度不同, 则必须将符干分开一个往上一个往下, 此时就不管在第几线了.

特殊的情况是, 如果同时演奏的两个音相差两度, 则必须左右错开写, 共用一个符干和符尾, 低音要写在左边.

从乐理可以知道, 上边的音符本身就代表了时值, 不过音符的长度是相对概念. 决定了一个音符实际长度的是BPM和拍号(Time signature)的共同结果.

搜了一下发现Unicode中对于音乐有完整的音乐符号, 下边就用HTML来显示一下完整的音符列表, 也为了以后复制方便:

  1. 倍全音符: 𝅜, double whole note
  2. 全音符: 𝅝, whole note
  3. 二分音符: 𝅗𝅥, half note
  4. 四分音符: 𝅘𝅥, quarter note
  5. 八分音符: 𝅘𝅥𝅮, eighth note
  6. 十六分音符: 𝅘𝅥𝅯, sixteenth note
  7. 三十二分音符: 𝅘𝅥𝅰, thirty-second note
  8. 六十四分音符: 𝅘𝅥𝅱, sixty-fourth note
  9. 一百二十八分音符: 𝅘𝅥𝅲, hundred twenty-eighth note

附点音符: dotted note, 就是在音符右侧下方加上一个点来改变音符的长度. 假如一个音符有n个附点, 实际长度是 (2^(n+1)-1)/(2^n) 倍 的原来音符长度.

比如有一个点的时候, 倍率 = (2^2-1)/(2^1) = 3/2 = 1.5倍, 则实际演奏的时候就是这个音符的1.5倍长度, 对于一个四分音符来说, 实际演奏一个四分音符加一个八分音符的长度.

有两个点的时候, 就是 (8-1)/4 = 7/4 = 1.75倍. 三个点的时候 = 15/8 = 1.875倍. 可见点越多, 越趋向于演奏成2倍原来的音.

连音 tuplet 和 花音 grace note

连音 Tuplet

连音记号用于把一段时间平均分割成普通音符无法表示的时间. 例如在4/4拍号下, 一拍内平均弹3个音. 很显然无法用八分音符表示这个音. 此时就需要把若干次数的音连接起来, 然后在上边标注要弹的次数.

最常见的连音是3:2和5:4的, 也就是4/4拍中一拍弹3个音或者一拍弹5个音.

而符号, 就要使用大于实际长度且最接近实际长度的音符来进行连音. 比如4/4一拍平均弹3个音, 最接近四分音符的1/3的就是八分音符(如果四分音符=1, 则实际音符=0.33, 八分音符=0.5, 十六分音符=0.25, 十六分音符小于0.33所以不能用). 如果弹5个音, 最接近四分音符且大于1/5的就是十六分音符(四分音符=1, 实际音符=0.2, 十六分音符=0.25, 八分音符-0.5, 都大于实际音符, 但十六分音符最接近.).

连音符号如下:

  1. 八分音符连音符号: 𝅪
  2. 十六分音符连音符号: 𝅫
  3. 三十二分音符连音符号: 𝅬

休止符 rest

休止符对应5种主要音符的休止符如下:

  1. 全休止符: 𝄻, whole note
  2. 二分休止符: 𝄼, half note
  3. 四分休止符: 𝄽, quarter note
  4. 八分休止符: 𝄾, eighth note
  5. 十六分休止符: 𝄿, sixteenth note

多小节休止符是这个样子: 𝄩, 上边会标数字, 表示休止多少个小节.

五线谱 Five-line Staff - 线, 间 与 谱号

音符终于看完了, 该到五线谱了.

五线谱又五线四间, 线和间都可以放音符. 可以上下加线, 加的线叫做 ledger line. 线越靠上, 音越高.

但是五线谱的线和间并没有对应到具体的某个音, 必须用谱号=clef来设定线对应的音.

最常用的是G谱号: 𝄞和 F谱号: 𝄢.

G谱号的意思是, 通过其肚子的那根线要当做中央C往上的G. 如果G谱号被放在第二线(肚子对准第二线), 就叫做高音谱号=Treble Clef.

所以这暗示着G谱号不一定被放在第二线, 不过如今的乐谱基本见不到第二线以外的G谱号了.

F谱号类似, 其两个点中间的那根线, 对应着中央C下边的F. 一般F谱号对应第四线(第四线是中央C下边的F), 此时叫做低音谱号=Bass Clef

除此之外, 还有一种叫做C谱号: 𝄡, 通过C谱号正中间的就是中央C.

这些谱号都可以放在任意一条线上. 所以谱号放到线上之后, 形成的是绝对对应关系, 即直接标明了一条线或者间与钢琴上一个键的对应关系.

还有一种谱号专门用于没有音高的乐器的谱号, 比如打击乐器, 叫做打击谱号: 𝄥.

所有的谱号在上方或者下方可以写8或者15, 如𝄟 𝄤,表示整个要高/低一个八度或者两个八度, 注意不是8和16, 是8和15, 8就表示一个八度, 15表示两个八度, 不要问为什么, 是规矩.

升降记号 sharp/flat sign

首先是符号本身的意思, ♯ 叫做升记号sharp, 高一个半音. ♭ 是降记号flat, 表示降半音. 还有一种还原记号natural ♮, 表示不升也不降.

这些记号需要搭配具体哪个音来使用. 如果写英文, 升降记号需要放在英文之后, 比如E-flat, G-sharp, 写成音名就是Eb G#, 中文会说降E升G.

还有低两个半音的重降记号 double flat: 𝄫 以及高两个半音的重升记号double sharp: 𝄪. 用法与单独的一样.

临时记号与调号

升降记号的核心问题是写在哪里. 如果直接写在谱子里的, 就是临时记号=accidental, 效果只有当下小节的同一个音(注意是同一个音, 而不是同一个音名, 比如升了C5, 不意味着C6也要升.)才有用. 过了这个小节就失效. 临时记号一定写在音符的正左边, 和音符对齐.

升降记号写在谱号后边, 拍号之前, 是永久性的, 影响全部乐谱. 此时就不称为升降记号, 而称为调号=key signature.

从乐理可以知道, 升或者降CDEFGAB中的某些音, 就是大调或者小调, 即调号会改变. 由于五线谱是绝对对应关系, 高音谱号, 下加一线固定对应物理琴键中央C, 那么所有的间和线都是白键. 升降记号标在哪一个线或者间, 就对应的升或者降那一个音名. 一定要注意是升降那一个音名, 假如升记号标在对应F的线上, 则所有的F, 全部都要升.

作为调号的时候, 标记有一定的顺序, 不能更改, 可以参见wiki的调号,

比如D大调需要升C和升F, 不能先标记C(三间C), 再标记F线, 是固定的必须先标记F线, 再是C.

拍号 Time signature 与 节拍速度 BPM

前边简单的提了一下拍号, 拍号写成上下的两个数字, 在文字里经常写成3/4这种.

上边的数字表示一小节有几拍, 下边的数字表示哪个音符为一拍. 结合BPM后就可以得到每个音符的绝对时值. 音乐的快慢就来自于拍号和BPM(也称为tempo)的共同作用.

这里注意有两种符号, 是一种简写:

  1. 𝄴, 这个就是4/4拍的简写
  2. 𝄵, 这个就是2/2拍的简写

BPM就是一分钟有多少拍, 一般是搭配拍号来使用, 比如规定了4/4之后, 就会再标记𝅘𝅥=120的样子, 就表示一分钟有多少拍, 即一分钟要演奏多少个四分音符的速度.

小节线 Bar

普通的小节线就是为了明确区分, 一般每小节中的拍数, 是与拍号的规定相等的. 小节线有如下几种:

  1. 𝄀: 普通小节线, 用于标注拍子的循环.
  2. 𝄁: 双小节线, 用于表示分段或者换调号.
  3. 𝄂: final bar, 表示整首曲子结束.
  4. 𝄆和𝄇: 成对出现, 表示范围内的谱子需要演奏两次, 也称为反复(重复)记号. 如果左边的反复记号没有出现, 则表示从头到右反复记号的部分演奏两次.

小节还有一些辅助记号:

  1. 𝄊 = Da capo = 从头开始
  2. 𝄉 = Dal segno = 从𝄋开始
  3. Fine = 结束

DC和DS一般和Fine一起用, 表示满足DC或者DS的条件后, 再次到这里就结束.

小节上边还会标有带有数字的框框, 和反复记号一起使用, 表示第一次演奏就演奏1号框中的东西, 第二次演奏要跳过1号框的时候继续演奏. 英文叫做First Ending, Second Ending.

小节中如果仅标记一个𝄎, 表示重复前边一个小节.

线

弧线

弧线分为连接线和圆滑线

弧线连接同音和仅仅只有两个音, 表示只要弹一个音, 中间不松开按键. 注意必须同音而且紧挨着, 只要跨越了, 就不是连接线.

所以三个相同音符第一个和第二个, 第二个和第三个各用弧线连接, 依然只弹一个音. 直接把第一个和第三个连接起来, 要弹三个音.

不是连接线的弧线, 都称为圆滑线, 不管连接相同音还是不同音. 圆滑线表示音不要断, 实际弹奏中一般是弹下一个音才放开上一个音.

锯齿线

锯齿线𝆱𝆲指的是滑音, 表示连续滑过键盘.

垂直的锯齿线𝆃会标记在几个音侧面, 表示琶音, 即快速弹完这几个音, 没有箭头的话表示从下到上, 带有下箭头𝆄就是从上到下.

在小节的上下标记带有8和15的虚线, 表示小节内的内容需要移高和移低一或者两个八度.
