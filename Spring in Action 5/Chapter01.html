Spring简介和术语

Spring核心提供了一个container，被称为Spring application context，用于创建和管理所有的组件。组件被称为beans。装配beans采用的模式叫做dependency injecttion，依赖注入。

历史上多用XML配置，在最近的版本中多使用Java配置。但目前，显式的XML和Java配置只有在Spring无法自动配置和发现组件的时候才使用。Spring自动发现组件的功能叫做autowiring和component scanning。

Spring Boot对于Spring的增强是autoconfiguration，节约了大量开发时间。

初始化 Spring 应用

Spring in Action 5把Spring 和Spring Boot一起说了，只有在特殊需要区分的时候才区分，可见Spring Boot现在已经是主流了。

初始化Spring应用的方法是Spring Initializr，这是一个Web应用，也是一个API。有很多种方法可以启动：

  1. http://start.spring.io
  2. curl
  3. Spring Boot command-line interface
  4. Spring Tool Suite
  5. IntelliJ IDEA
  6. NetBeans

Spring Tool Suite是一个工具，用于一些IDEA来增强Spring开发，作者说还有一些其他IDE没有的内容。反正我还是使用IntelliJ。新建项目的时候选择Spring Intializr，然后会到网站读取配置信息，之后输入即可，一些内容已经都知道了，starter里选上Devtools，Web和Thymeleaf。
